# Minecraft-1.6.4-MDK

This repository provides a modernized and repaired Minecraft 1.6.4 Mod Development Kit (MDK) built on a **fixed version of ForgeGradle 1.0**, enabling the project to compile successfully under **Gradle 4.x**.

The modified ForgeGradle 1.0 source code used in this MDK can be found here:  
[SourceCode](https://github.com/zeng-github01/ForgeGradle/tree/FG_1.0-fix)


This setup is intended for developers who want a **stable and reproducible environment** for legacy Minecraft 1.6.4 modding without relying on outdated or broken build systems.
